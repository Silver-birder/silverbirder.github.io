これまで、ブログへのコメントサービスに[Giscus](https://giscus.app/ja)をブログサイトに設置していました。
本日、同コメントサービスの[Disqus](https://disqus.com/)に切り替えました。

Giscusの場合、GitHubアカウントが必須だったのですが、DisqusならGoogleやXなどのアカウントでも可能となるからです。
これで、非エンジニアでもコメントできるようになります！はろー！
